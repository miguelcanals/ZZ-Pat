# ZZ-Pat

In ZZ-docs you will find:

- ZZ-PatFlow.odt -> Gives you an idea of the ZZ-PaT flow
- ZZ-Pat-Server-Install.odt -> Server installation
- ZZ-Pat-Client-install.odt -> Client install
